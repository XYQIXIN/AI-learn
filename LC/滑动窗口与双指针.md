# 一：定长滑动窗口

---

## 核心框架：三步法

窗口右端点在 `i` 时，由于窗口长度为 `k`，所以窗口左端点为 `i − k + 1`。

整个过程总结为：**入 → 更新 → 出**

| 步骤 | 操作 | 说明 |
|------|------|------|
| **入** | 下标 `i` 的元素进入窗口 | 更新相关统计量。若左端点 `i−k+1 < 0`，窗口尚未形成，继续入 |
| **更新** | 更新答案 | 一般是更新最大值/最小值 |
| **出** | 下标 `i−k+1` 的元素离开窗口 | 更新相关统计量，为下一轮循环做准备 |

## 题目 1：长度为 K 的子串最大元音数

**问题**：给你字符串 `s` 和整数 `k`，请返回字符串 `s` 中长度为 `k` 的单个子字符串中可能包含的最大元音字母数。

```python
class Solution:
    def maxVowels(self, s: str, k: int) -> int:
        ans = vowel = 0
        for i, c in enumerate(s):
            # 1. 入：当前字符是元音则计数+1
            if c in "aeiou":
                vowel += 1

            left = i - k + 1
            if left < 0:
                continue

            # 2. 更新：记录最大元音数
            ans = max(ans, vowel)
            if ans == k:          # 早停优化：已达到上限
                break

            # 3. 出：左端点移出窗口，若是元音则计数-1
            if s[left] in "aeiou":
                vowel -= 1

        return ans
```

## 题目 2：长度为 K 的子数组最大平均数

**问题**：给你一个由 `n` 个元素组成的整数数组 `nums` 和一个整数 `k`，请你找出平均数最大且长度为 `k` 的连续子数组，并输出该最大平均数。

```python
class Solution:
    def findMaxAverage(self, nums: List[int], k: int) -> float:
        max_s = -inf
        s = 0
        for i, c in enumerate(nums):
            # 1. 入
            s += c

            left = i - k + 1
            if left < 0:
                continue

            # 2. 更新
            max_s = max(max_s, s)

            # 3. 出
            s -= nums[i - k + 1]

        return max_s / k
```

## 题目 3：半径为 K 的子数组平均值

**问题**：给定数组 `nums` 和整数 `k`。半径为 `k` 的子数组平均值指：以 `i` 为中心、半径 `k` 内所有元素的平均值（范围 `[i−k, i+k]`）。如果不足 `k` 个元素则返回 `-1`。返回长度为 `n` 的数组 `avgs`。

**思路**：本题相当于一个长为 `2k + 1` 的滑动窗口。

| 步骤 | 操作 |
|------|------|
| **入** | 下标 `i` 的元素进入窗口，和 `s` 增加 `nums[i]`。若 `i < 2k` 则重复 |
| **更新** | 记录答案 `avgs[i−k] = ⌊s / (2k+1)⌋`（`i−k` 是子数组正中间） |
| **出** | 下标 `i−2k` 的元素离开窗口，和 `s` 减少 `nums[i−2k]` |

```python
class Solution:
    def getAverages(self, nums: List[int], k: int) -> List[int]:
        avgs = [-1] * len(nums)  # 占位数组：-1 表示无法形成完整窗口
        s = 0                     # 窗口元素和

        for i, x in enumerate(nums):
            # 1. 入
            s += x

            if i < 2 * k:         # 窗口大小不足 2k+1
                continue

            # 2. 更新：记录在子数组中间位置 i-k
            avgs[i - k] = s // (2 * k + 1)

            # 3. 出
            s -= nums[i - 2 * k]

        return avgs
```

## 题目 4：长度为 K 且平均值 ≥ threshold 的子数组数目

**问题**：给定整数数组 `arr` 和整数 `k`、`threshold`，返回长度为 `k` 且平均值 ≥ `threshold` 的子数组数目。

```python
class Solution:
    def numOfSubarrays(self, arr: List[int], k: int, threshold: int) -> int:
        s = 0    # 窗口元素和
        ans = 0  # 满足条件的子数组数目

        for i, x in enumerate(arr):
            # 1. 入
            s += x

            if i < k - 1:         # 窗口尚未形成
                continue

            # 2. 更新：避免浮点数，用 s >= k * threshold
            if s >= k * threshold:
                ans += 1

            # 3. 出
            s -= arr[i - k + 1]

        return ans
```

## 题目 5：得到连续 K 个黑色块的最少操作次数

**问题**：字符串 `blocks` 中 `W`=白色、`B`=黑色。每次可把一个白色块涂黑，返回出现连续 `k` 个黑色块的**最少**操作次数。

```python
class Solution:
    def minimumRecolors(self, blocks: str, k: int) -> int:
        count = 0     # 当前窗口中 W 的数量（需涂黑的次数）
        result = k    # 最多需要 k 次（全白的情况）

        for i, x in enumerate(blocks):
            # 1. 入
            if x == 'W':
                count += 1

            left = i - k + 1
            if left < 0:          # 窗口尚未形成
                continue

            # 2. 更新
            result = min(result, count)

            # 3. 出
            if blocks[left] == 'W':
                count -= 1

        return result
```

题目6 给你一个整数数组 nums 和两个正整数 m 和 k 。
请你返回 nums 中长度为 k 的 几乎唯一 子数组的 最大和 ，如果不存在几乎唯一子数组，请你返回 0 。
如果 nums 的一个子数组有至少 m 个互不相同的元素，我们称它是 几乎唯一 子数组。
子数组指的是一个数组中一段连续 非空 的元素序列。

class Solution:
    def maxSum(self, nums: List[int], m: int, k: int) -> int:
        ans = s = 0
        cnt = defaultdict(int)
        for i, x in enumerate(nums):
            s += x
            cnt[x] += 1
            left = i - k + 1
            if left < 0:
                continue
            if len(cnt) >= m:
                ans = max(ans, s)
            out = nums[left]
            s -= out
            cnt[out] -= 1
            if cnt[out] == 0:
                del cnt[out]
        return ans
