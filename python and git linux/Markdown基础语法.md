# Markdown 基础语法指南

## 1. 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

## 2. 文本格式

```markdown
**粗体**
*斜体*
~~删除线~~
`行内代码`
```

**效果**：
- **粗体**
- *斜体*
- ~~删除线~~
- `代码`

## 3. 列表

### 无序列表
```markdown
- 苹果
- 香蕉
  - 子项（缩进）
```

### 有序列表
```markdown
1. 第一项
2. 第二项
3. 第三项
```

## 4. 链接和图片

```markdown
[链接文字](https://example.com)
![图片描述](图片链接)
```

## 5. 代码块

```python
# 三个反引号 + 语言
def hello():
    print("Hello")
```

## 6. 表格

```markdown
| 表头1 | 表头2 |
|-------|-------|
| 内容1 | 内容2 |
| 内容3 | 内容4 |
```

## 7. 引用

```markdown
> 这是引用内容
> 可以多行
```

> 这是引用效果

## 8. 分割线

```markdown
---
```

---

## 9. 复选框（任务列表）

```markdown
- [ ] 未完成
- [x] 已完成
```

- [ ] 未完成
- [x] 已完成

## 10. 常用技巧

| 语法 | 说明 |
|------|------|
| `>` | 引用 |
| `- [ ]` | 复选框 |
| `---` | 分割线 |
| `` ` `` | 行内代码 |
| `Ctrl + K` | 快速插入链接（VSCode）|
